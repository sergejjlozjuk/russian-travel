### Приложение 'Путешествие по России'. Front-end 


## Интро
Здесь будет проект о путешествии по России. Проект создан для ознакомления с красивейшими местами России.

## Технологии
* Flex-box
* Grid
* Резиновая верстка
* БЭМ
* Медиа-запрос
